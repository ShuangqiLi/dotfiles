# Fonts — MesloLGS NF

These files come from [romkatv/powerlevel10k-media](https://github.com/romkatv/powerlevel10k-media) (same family Powerlevel10k documents for Powerline-style / icon glyphs).

## Committed in this repo

- **`MesloLGS NF *.ttf`** (four faces) — **commit them** after download so clones work offline.
- **`MesloLGS NF License.txt`** — Apache 2.0; keep it alongside the fonts when redistributing.

`fonts/.uuid` is still ignored (generated by fontconfig).

## If the TTFs are not in your clone yet

On a machine that already downloaded them (or after a successful `bash scripts/install-meslo-fonts.sh`):

```bash
git add fonts/MesloLGS\ NF\ *.ttf fonts/MesloLGS\ NF\ License.txt
git commit -m "Add MesloLGS NF fonts"
```

## `./install` behavior

`scripts/install-meslo-fonts.sh` **does not use the network**. It checks that the four TTFs are present under `fonts/` (they should come from git). To copy from another directory on the machine: `MESLO_FONT_SOURCE_DIR=/path/to/fonts bash scripts/install-meslo-fonts.sh`.

To skip the step entirely (e.g. CI): `SKIP_MESLO_FONTS=1 ./install`.

## Server vs IDE

| Where | Role |
|--------|------|
| **`fonts/`** + `~/.fonts` + `fc-cache` | Linux GUI terminals on the server (fontconfig). |
| **Cursor / VS Code (your PC)** | Integrated terminal still needs **MesloLGS NF** installed or selected in `terminal.integrated.fontFamily` — see root **README.md**. |
